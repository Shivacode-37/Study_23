## Work flow suggestion
# Heading
Makes a big title.
## Heading
Makes a medium subtitle.
### Heading
Makes a small section title.
Bold → **text**
Makes important text bold.
Italic → *text*
Makes text slanted for light emphasis.
Bold + Italic → ***text***
Makes very strong emphasis.
Blockquote → > text
Adds a highlighted note box.
Bullet List → - item or *
Creates a clean bullet point list.
Number List → 1. item
Creates a step-by-step numbered list.
Inline Code → `text`
Highlights small code words.
Code Block → triple backticks
Shows multi-line code with syntax colors.
Horizontal Line → ---
Creates a visual divider between sections.
Checkbox → - [ ] or - [x]
Adds tasks or checklist items.
Links → [name](url or file.md)
Creates clickable links to notes or websites.
Table → | col1 | col2 |
